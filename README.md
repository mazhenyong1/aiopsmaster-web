# AIOps Master Web

Welcome to the **AIOps Master** demo website! This repository hosts a **public static website** showcasing the AIOps Master platform, an AI-driven DevOps & SRE orchestration system.

## About

This website provides an overview of the platform, its features, and demo visualizations. It's designed as a **public-facing introduction** to the AIOps Master project.

- **Platform:** AI-powered automation for IT operations
- **Features Highlighted:**
  - Event-driven decision engine
  - AI-assisted CI/CD and SRE automation
  - Real-time dashboard and monitoring
  - Executor automation with Ansible / Docker
  - ELK-based logging and AI anomaly detection

## Note on Repository Visibility

- This **web repository is public** and meant for demonstration purposes.  
- The main **AIOps Engine code repository (`aiops-engine`) is private**, containing the full platform source code, AI agents, and sensitive infrastructure logic.
